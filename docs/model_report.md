# Model Performance

Models are generated by:

```powershell
$env:PYTHONPATH="$PWD\backend;$PWD"
python -m ml.training.train --dataset datasets/phishing_urls_seed.csv --model-dir ml/models --metrics ml/models/metrics.json
```

Metrics written to `ml/models/metrics.json`:

- accuracy
- precision
- recall
- F1 score
- ROC-AUC

The repository includes a seed dataset for local bootstrapping. For stronger production metrics, run `scripts/download_feeds.py` to pull OpenPhish, URLHaus, and PhishTank data, then retrain with `datasets/phishing_urls_training.csv`.
